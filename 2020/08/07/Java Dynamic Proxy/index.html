代理模式

在Java中，代理被分为静态代理以及动态代理，动态代理模式是通过反射来实现的，静态代理模式与动态代理模式的区别有：

  1. 静态代理模式代理类在代码运行之前就生成.class文件了，而动态代理模式代理类是在代码运行时由JVM动态创建代理类
  2. 静态代理模式代理类需要提前实现接口编写代理类，动态代理模式不需要提前实现接口编写代理类

    1.1 动态代理的实现方式

    动态代理的实现需要用到Java包下面的两个类：Java.lang.reflect.InvocationHandler 接口和 Java.lang.reflect.Proxy类
